本文分享如何从零开始搭建Linux服务器

前期准备工作

  • 域名解析
  • linux服务器安装nginx

域名解析

购买域名

本文案例在阿里云上购买的域名

后续跟着步骤走就行了

域名解析

在 控制台->域名->解析

点开之后，接着 添加记录

打开之后会有以下几个选项

记录类型常用主要有两种：

  • A: 指向服务器的IP地址
  • CNAME: 指向其他域名

主机记录：可添加www或者不填也行（一般记录两条信息，其中之一填www，另一个不填； 这样无论在域名前面加www都能正常访问 ）

记录值： 当你用 A 记录类型的，应当填服务器外网IP地址；使用 CNAME 记录类型，填写域名地址

添加完之后，一般等待十分钟后就能正常访问了

Linux服务器安装nginx

本文示例云服务器：阿里云9.9元/月学生机

  • 系统：centOS7.x

本文是基于centOS安装，不同的服务器的安装方式是不一样哦

nginx安装准备

接下来在安装nginx前，先安装好编译环境，以及工具

安装make

yum -y install gcc automake autoconf libtool make

安装g++

yum install gcc gcc-c++

编译环境已经安装完成，下面将会安装一些库，一般都会安装在
/usr/local/src

进入当前目录
cd /usr/local/src

安装PCRE库

wget ftp://ftp.csx.cam.ac.uk/pub/software/programming/pcre/pcre2-10.31.tar.gz

目前最新版本是10.31，如果想查看最新版本可以点击这里

只需将安装包名替换掉就可以了

接着，解压并安装pcre

(每一行对应一条指令)
tar -zxcf pcre2-10.31.tar.gz
cd pcre2-10.31
./configure
make
make install

安装zlib库

wget http://zlib.net/zlib-1.2.11.tar.gz

可以查看最新版本

解压安装

tar -zxvf zlib-1.2.11.tar.gz
cd zlib-1.2.11
./configure
make
make install

安装ssl

wget https://www.openssl.org/source/openssl-1.1.1b.tar.gz
tar -zxvf openssl-1.1.1b.tar.gz

好了，到了这里， 我们开始进入主题吧

安装nginx

安装repository管理及扩展包的工具

yum install yum-utils

创建一个文件，并把下面内容填进去

vim /etc/yum.repos.d/nginx.repo

[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key
[nginx-mainline]
name=nginx mainline repo
baseurl=http://nginx.org/packages/mainline/centos/$releasever/$basearch/
gpgcheck=1
enabled=0
gpgkey=https://nginx.org/keys/nginx_signing.key

如果想使用主线版的nginx，请输入下面命令

sudo yum-config-manager --enable nginx-mainline

下面就开始安装nginx

yum install nginx

过几分钟后，在浏览器输入http://IP就可以访问了

介绍几个有关nginx常用的命令

whereis nginx // 查看目录结构

nestat -ntpl | grep 80 // 查看端口号是否占用

nginx  // 启动服务器

nginx -s stop  // 关闭服务器

nginx -s reload  // 重启服务器

nginx -s /etc/nginx/nginx.conf  // 使用指定的配置文件启动

nginx -v  // 查看版本信息

nginx -t // 测试配置文件是否有错误

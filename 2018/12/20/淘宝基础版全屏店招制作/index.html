淘宝店招 css

今天在工作上遇到一些问题：

如何实现淘宝店铺基础版的全屏店招？

百度了很多遍方法，也找不出好的解决办法。
只好实践出真理了。

首先，在淘宝基础版中，店招最大的宽度950px，高度120px~150px。而页头背景也只有固定的蓝白色。如果想要实现自定义的店招，就只能通过css来处理了。

第一步，用ps制作好全屏店招。这制作过程省略……
（本人顺便用一个海报来做演示）

第二步，将店招图片上传到淘宝的图片空间

第三步，进入淘宝装修后台，点开导航条的右上角的设置，在显示设置里添加css代码。


.skin-box-bd {
    position: relative;
    left: -485px;
    top: -120px;
    width: 1920px;
    height: 150px;
    padding-left: 485px;
    padding-top: 120px;
    background: url("店招图片背景地址") 0 0 no-repeat;
}

第四步，点击发布就OK了，就可以看到全屏店招了

当然，这全屏店招有些限制，比如

  1. 不能添加热点连接
  2. 不能放置搜索框

如果以后发现有这些方面的优化，我也会第一时间来更新。

2019/1/11

今天终于解决如何添加热点链接的问题了

首先，我们需要自定义店牌，也就是说通过代码来实现店牌。

废话不多说，直接上代码：

<div style="1920px; height: 150px;">

    <div class="footer-more-trigger vjs-center-container" style="width: 1px; height: 1px; border: 0; padding: 0; line-height:0; left: auto; top: auto; background: none;margin:0">

        <div class="footer-more-trigger vjs-center-container" style="width:1920px; height: 150px; border: 0; padding: 0; line-height:0; left: -485px; top: auto; background: none;margin:0; z-index: 99;">

            <img src="店招图片地址" />
        </div><!-- 居中显示 -->
    </div><!-- 定位参考 -->
</div><!-- 容器 -->

紧接着，设置导航，在显示设置添加下面代码


.skin-box-bd {
    position: relative;
    top: -150px;
    left: -485px;
    padding-left: 485px;
    padding-top: 150px;
    background: url("图片地址");
    width: 1920px;
}

这里有几点需要注意下：

  1. 第二个和第三个div的class属性，一定要添加vjs-center-container，否则在宝贝详细页就会点击不了链接（本人测试过）

  2. .skin-box-hd 一定要添加width: 1920px;否则就会显示950px宽，并居左显示。

通过这个方法，不仅可以添加链接，还可以制作一些别的交互，比如表单、优惠提示等等！
